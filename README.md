# ERP系统 v4.4

一个轻量级、功能完整的进销存管理系统，支持全平台访问（Windows/Mac/iOS/Android）和 Docker 部署。

## ✨ 主要功能

### 📦 库存管理
- **多仓库管理**：支持多个仓库，每个仓库独立管理库存
- **默认仓库切换**：一键切换默认仓库
- **仓位管理**：精细化库存管理，支持仓库内多个仓位（货架/货位）
- **多仓位SKU**：同一商品可存放在多个仓位，独立统计库存
- **库存调拨**：支持仓库间、仓位间的库存调拨
- **入库管理**：支持批量入库、加权平均成本计算
- **快速补货** ⭐：一键补货，自动填充商品和仓位信息
- **库龄统计**：自动计算商品库龄，便于库存周转分析
- **智能导入** ⭐：支持 Excel 批量导入，自动创建仓库和仓位并入库
- **表头排序** ⭐ NEW：库存列表支持点击表头排序（品牌/名称/仓库/价格/库存/库龄）
- **移动端卡片视图** ⭐ NEW：手机端库存页面采用卡片布局，无需横向滚动

### 🛒 销售管理
- **多种销售类型**：
  - 现款销售：即时收款，自动结清
  - 账期销售：记录应收账款
  - 寄售调拨：商品调拨到寄售仓
  - 寄售结算：寄售商品结算
  - 退货：关联原始销售订单退货
- **智能退货**：
  - 必须关联原始销售订单
  - 自动计算可退数量（已退数量统计）
  - 防止超额退货
  - 支持部分退货
  - 支持退款/形成在账资金
- **一单多仓出库** ⭐：同一订单可从多个仓库/仓位出库
- **智能库存提示** ⭐：下拉框实时显示各仓库/仓位库存
- **仓位级销售**：销售和退货可指定具体仓位
- **实时库存查询**：按仓库、仓位筛选商品

### 🚚 物流管理
- **物流跟踪**：销售订单自动生成物流记录，集中管理所有发货状态
- **快递100集成**：接入快递100 API，实时查询物流轨迹
- **一单多件** ⭐：同一订单支持多个快递单号（多仓发货场景）
- **状态自动更新**：支持实时查询和推送回调两种方式更新物流状态
- **快递单号复制**：一键复制所有快递信息（含物流公司名称和SN码），方便发给客户
- **11家快递公司**：内置顺丰、圆通、中通、韵达、申通、EMS、京东、极兔、德邦、中国邮政 + 上门自提
- **SN码管理** ⭐ NEW：物流单支持录入SN码（序列号），用于产品管控追踪，支持批量录入
- **上门自提** ⭐ NEW：客户上门自提无需快递单号，状态直接变为已签收
- **发货地址显示** ⭐ NEW：物流详情页直接显示订单备注中的发货地址
- **表头排序** ⭐ NEW：物流列表支持点击表头排序（订单号/类型/客户/快递/状态）

### 📦 寄售管理
- **每客户独立寄售仓** ⭐ NEW：每位客户自动创建独立虚拟仓库，清晰管理寄售库存
- **就地寄售结算**：直接从客户虚拟仓结算，无需调拨回实体仓
- **寄售退货**：支持从虚拟仓退回实体仓，部分退货
- **寄售汇总**：按客户汇总寄售金额和库存
- **虚拟仓切换**：库存页面支持显示/隐藏虚拟仓库

### 🛍️ 采购管理 ⭐ NEW
- **供应商管理**：供应商档案维护（名称、联系人、电话、税号、银行账户）
- **采购订单**：创建采购订单、含税/不含税价格自动计算
- **模糊搜索** ⭐ NEW：供应商和商品支持模糊搜索选择，告别慢速下拉框
- **快速建档** ⭐ NEW：采购页面直接新建商品，无需切换到库存模块
- **采购付款**：财务确认付款，记录付款人和时间
- **采购取消** ⭐ NEW：待付款状态可取消采购单，订单完结
- **采购收货**：按明细逐项收货，支持部分收货、分批到货
- **自动入库**：收货自动更新库存，加权平均成本计算
- **状态追踪**：pending → paid → partial → completed / cancelled 完整状态流转

### 💰 财务管理
- **返利管理** ⭐ NEW：客户/供应商返利余额充值、销售/采购订单逐行返利抵扣
- **返利充值**：财务新增充值（下拉选择对象）或行内充值，完整流水记录
- **返利抵扣**：销售订单确认时勾选使用返利，逐行输入返利金额，自动校验余额
- **采购返利**：采购订单使用返利，收货入库成本自动降低（返利后有效单价）
- **在账资金管理**：退货可选择退款或形成在账资金，销售自动抵扣
- **记账凭证**：从销售/采购/收款单据一键生成标准记账凭证
- **凭证打印**：标准凭证格式预览（241×139.5mm二等分纸），支持打印和下载PDF
- **税率选择**：凭证生成时可选择增值税税率（13%/9%/6%/3%/免税）
- **大写金额**：凭证合计行自动显示中文大写金额
- **应收管理**：收款记录关联销售订单，点击整行查看订单详情，桌面端表格视图
- **应付管理**：采购应付款集中管理，确认/取消付款，点击查看采购详情
- **欠款明细**：桌面端表格视图，点击直接查看订单详情
- **付款方式详情**：订单详情显示每笔收款的具体付款方式（现金/转账/支付宝等）
- **收款确认机制**：财务人员可确认到账，记录确认人和时间
- **可配置收款方式**：支持自定义增删改收款方式（财务设置页面）
- **应收账款**：自动统计客户欠款
- **订单追溯**：退货单可追溯到原始销售订单
- **盈利分析**：毛利统计、成本分析（退货自动扣减）
- **出入库日志**：完整的库存变动记录（PURCHASE_IN显示中文"采购入库"）
- **订单导出**：支持导出Excel订单明细
- **订单搜索**：支持按订单号、客户名、SN码、快递单号模糊搜索
- **表头排序**：财务订单和出入库日志支持点击表头排序

### 📋 SN码管理 ⭐ NEW
- **SN配置**：按"仓库+品牌"组合启用SN码管理
- **入库录入**：入库/采购收货时检测SN需求，必填SN码（textarea批量输入）
- **发货扣减**：物流发货时SN码从池中扣减（in_stock → shipped）
- **库存导出**：导出Excel含SN码数量列
- **全链路追踪**：入库→库存→发货完整SN生命周期管理

### 👥 客户管理
- **客户档案**：客户基本信息维护
- **交易记录**：查看客户所有交易明细
- **余额管理**：
  - 欠款统计（正数显示）
  - 在账资金统计（负数显示）
  - 颜色区分：红色=欠款，蓝色=在账资金
- **月度汇总**：按月份查看客户交易统计

### 🔐 权限管理
- **角色管理**：管理员、普通用户
- **11项细粒度权限** ⭐ NEW：dashboard、销售、物流、寄售、库存查看/操作、客户、财务查看/确认收款、日志、系统设置
- **用户管理**：新增、编辑、禁用用户
- **修改密码**：所有用户均可在系统设置中自行修改密码
- **操作日志** ⭐ NEW：记录8类关键操作（创建订单、收款、入库、调拨等），按类型筛选查看
- **自动退出**：4小时无操作自动退出登录，防止设备丢失导致数据泄露
- **登录限流**：5分钟内最多5次登录尝试，防暴力破解

### 🛡️ 安全与备份
- **自动备份**：每天凌晨3点自动备份数据库，保留30天

- **启动备份**：每次服务启动自动创建备份
- **原子操作**：库存和余额变更使用数据库原子操作，防止并发冲突
- **防重复提交**：订单提交按钮防重复点击
- **CSV注入防护** ⭐ NEW：导出CSV时自动转义危险前缀字符
- **登录限流内存保护** ⭐ NEW：IP记录上限10000，防止内存泄漏
- **SECRET_KEY安全**：未设置环境变量时自动生成随机密钥
- **CDN版本锁定**：前端依赖库版本锁定，防止供应链攻击
- **401自动登出**：Token过期时自动退出并提示
- **密码强度**：密码最少6位
- **上传限制**：文件上传最大10MB
- **错误脱敏**：API错误信息不泄露内部细节

## 🚀 快速开始

### 方式一：Windows 本地部署（开发/测试）

#### 1. 安装依赖

```bash
# Python 3.11+
pip install -r requirements.txt
```

#### 2. 启动服务

**本机访问（仅本机）：**
```bash
python -m uvicorn main:app --host 127.0.0.1 --port 8000
```

**局域网访问（推荐 - 支持手机/平板）：**
```bash
python -m uvicorn main:app --host 0.0.0.0 --port 8000
```

#### 3. 访问系统

**本机访问：**
- 打开浏览器访问：`http://127.0.0.1:8000`

**局域网访问（手机/平板/其他电脑）：**
- Windows：查看本机IP - 运行 `ipconfig`
- Mac：查看本机IP - 运行 `ifconfig | grep inet`
- 使用 `http://本机IP:8000` 访问（例如：`http://192.168.1.100:8000`）

**当前服务器IP：**
- 局域网地址：`http://192.168.124.225:8000`
- 手机用户：确保手机和电脑连接同一个Wi-Fi，然后用手机浏览器访问上面的地址

## 📊 功能亮点

### 1. 精细化库存管理

支持三级库存管理：**仓库 → 仓位 → 商品**

```
实体仓库
├── A-01 货架
│   ├── 耐克运动鞋 (10件)
│   └── 阿迪运动鞋 (5件)
├── A-02 货架
│   └── 耐克运动鞋 (8件)
└── B-01 货架
    └── 彪马运动鞋 (12件)
```

### 2. 智能退货控制

- ✅ 必须关联原始销售订单
- ✅ 只能退原订单中的商品
- ✅ 自动计算可退数量
- ✅ 防止超额退货

```
销售：A商品 10件
退货1：A商品 3件 → 剩余可退 7件
退货2：A商品 7件 → 剩余可退 0件
退货3：无法继续退货（已全部退完）
```

### 3. 一单多仓出库 ⭐ NEW

一张订单可以从多个仓库/仓位出库：

```
订单 #2024001
├── 商品A × 10件 → 从仓库A-A-01出库
├── 商品B × 5件  → 从仓库B-B-02出库
└── 商品C × 8件  → 从仓库A-A-03出库
```

**智能库存提示**：
- 仓库下拉框：`仓库A (库存: 50)`
- 仓位下拉框：`A-01 (库存: 20)`
- 实时颜色提示：绿色=充足，红色=不足

### 4. 快速补货 ⭐ NEW

库存表格每个商品后面都有"补货"按钮：

```
点击补货 → 自动填充商品/仓库/仓位 → 输入数量和成本 → 确认
   2秒完成，比原来的7步入库流程快3倍！
```

### 5. 库存调拨

支持仓库间、仓位间灵活调拨：

```
调出：实体仓库 A-01 → 调入：实体仓库 B-02
商品：耐克运动鞋
数量：5件
```

### 6. 加权平均成本

自动计算商品加权平均成本：

```
入库1：10件 @ ¥100 = 总成本 ¥1000
入库2：20件 @ ¥110 = 总成本 ¥2200
加权成本 = (1000 + 2200) / (10 + 20) = ¥106.67/件
```

### 7. 智能导入 ⭐ NEW

Excel 导入自动创建仓库、仓位并入库：

```excel
SKU    | 名称   | 仓库   | 仓位  | 数量
SKU001 | 商品1  | 新仓库  | A-01 | 50
```

**自动处理**：
1. 检测到"新仓库"不存在 → 自动创建
2. 检测到"A-01"不存在 → 自动创建
3. 自动入库50件，计算加权成本
4. 导入完成后前端自动刷新仓库列表

### 数据库备份

**自动备份（内置）**：
- 系统每天凌晨3点自动备份，保留最近30天
- 每次启动服务时自动创建备份
- 管理员可在"系统设置 → 备份管理"手动创建/下载/删除备份

**手动备份（命令行）**：
```bash
# 备份
cp ./data/erp.db ./data/erp.db.backup.$(date +%Y%m%d)

# 恢复
cp ./data/erp.db.backup.20260204 ./data/erp.db
docker-compose restart
```

ports:

```yaml

  - SECRET_KEY=your-secret-key-here     # JWT密钥（不设置则自动生成随机密钥）

  - CORS_ORIGINS=*                      # 允许的跨域来源（逗号分隔）
  - KD100_KEY=your-kd100-key            # 快递100 API Key
  - KD100_CUSTOMER=your-kd100-customer  # 快递100 Customer ID
  - KD100_CALLBACK_URL=http://your-domain/api/logistics/kd100/callback
```
